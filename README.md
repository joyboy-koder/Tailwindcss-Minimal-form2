# Tailwind Minimal Form 2

A clean and minimal form designed using **Tailwind CSS**. This form is fully responsive, easy to customize, and perfect for modern web projects.  

## Features

- Minimal and clean design
- Fully responsive for mobile and desktop
- Easy to customize with Tailwind CSS
- Lightweight and fast
- Accessible form inputs
